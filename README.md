## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migrations

```bash
# create migration
$ npm run migration:create --name=users-seed

# generate migration
$ npm run migration:generate --name=init

# run migration
$ npm run migration:run

# revert migration
$ npm run migration:run
```
